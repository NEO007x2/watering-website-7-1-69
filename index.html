Technical Specifications

Built with robust and reliable components

ESP32 Microcontroller

  • Dual-core Xtensa 32-bit LX6
  • 240 MHz clock speed
  • 520 KB SRAM
  • WiFi 802.11 b/g/n
  • Bluetooth 4.2 + BLE

ESP32-CAM Module

  • OV2640 2MP camera
  • JPEG/BMP/Grayscale output
  • Built-in flash LED
  • MicroSD card slot
  • Real-time streaming

Connectivity
